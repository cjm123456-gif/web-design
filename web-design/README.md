# 高校智能体应用展示网站

一个无构建依赖的静态产品展示网站，按照 `DESIGN.md` 中确认的黑紫未来教育视觉方案实现。

## 本地预览

在项目目录运行：

```powershell
python -m http.server 8000
```

然后在浏览器打开 `http://127.0.0.1:8000`。

网站由原生 HTML、CSS 和 JavaScript 构成，不需要安装 npm 依赖，可直接部署到任意静态网站服务。

## 文件说明

- `index.html`：页面内容、栏目结构和内联 SVG 图标。
- `styles.css`：颜色变量、响应式布局、动画和组件状态。
- `script.js`：浮动导航、焦点受控菜单、滚动渐显、数字动画、球面流体可见性管理、外圈 SVG 波面、智能体关系图指针视差、聚光卡片、即时按压与移动端成品横滑状态。
- `DESIGN.md`：完整设计规范。
- `AUDIT.md`：参考图差异与质量检查记录。

## 后续内容替换

在 `index.html` 中搜索以下文字即可快速定位待完善区域：

- `资料待补充`：替换项目名称、学校、截图、功能与成果。
- `更多成果，持续更新`：需要增加展示位时复制对应的项目卡片。

页面定位为纯智能体应用展示，不添加公司名称、企业 Logo、地区属性、资质、联系方式或企业宣传内容；也不要虚构项目数量、提效比例或评价。

## 替换成品截图

成品区当前使用的是结构化界面示意图，不是虚构项目截图。获得真实素材后，可以在对应的 `.project-card__visual` 区块中替换为：

```html
<img src="assets/项目截图.webp" alt="某高校某智能体的界面截图">
```

建议图片使用 WebP 或 AVIF，宽度不低于 1200px，并保留清晰、准确的替代文本。

## 字体说明

页面不加载外部字体，使用 Apple/system-first 字体栈：macOS 与 iOS 优先使用 SF Pro 和苹方，Windows 自动使用系统中文字体。页面可完全离线加载。
